# Assets Folder

Place your game assets in this folder:

## Required Files

1. **billy-sprite.png** - Billy's sprite sheet

   - Horizontal layout with 8 frames for running animation
   - Recommended size: 400x50 pixels (50x50 per frame)
   - Transparent background

2. **priest.png** - Priest obstacle image

   - Recommended size: 45x60 pixels
   - Transparent background

3. **bird.png** - Bird obstacle image

   - Recommended size: 40x30 pixels
   - Transparent background

4. **mouse.png** - Mouse obstacle image
   - Recommended size: 35x25 pixels
   - Transparent background

## Notes

- All images should be PNG format with transparent backgrounds
- The game will use placeholder graphics if images are not found
- You can test the game without assets to see the placeholders
